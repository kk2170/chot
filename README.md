# chot

**chot** は、チャットのように気軽に書けるメモアプリです。  
名前は **Chat + note** の合成語で、"ちょっと書く" というニュアンスも込めています。

## コンセプト

- チャットUIで、思いついたことをそのまま流すように記録する
- スマホでもデスクトップでも、同じ体験で使える
- メモの心理的ハードルを下げて、"ちょっと書く" を続けやすくする

## 想定する利用シーン

- ふと思いついたアイデアの記録
- 作業メモや日報の下書き
- 買い物メモやタスクの書き残し
- 自分との会話のようなログ保存

## MVPで目指したい機能

- 1つの画面でメモを送信できるチャットUI
- 時系列でメモが並ぶシンプルな表示
- メモの保存と再表示
- スマホ・デスクトップ両対応のレイアウト
- ローカル保存を前提にした軽い動作

## 今後追加を検討できる機能

- 検索
- タグ付け
- ピン留め
- 日付ごとの振り返り
- クラウド同期
- 複数スレッド / 複数ノート

## 技術方針

- **Flutter** で開発
- 単一コードベースでモバイルとデスクトップに対応
- 初期段階では、まず使い心地のよいUIと最小機能を優先

## 現在の実装内容

- チャット風の1画面UI
- 入力したメモを時系列で表示
- `shared_preferences` によるローカル保存
- 複数ノートの作成・切り替え・タイトル変更・削除
- Slack風のノート一覧UIと、投稿ごとのスレッド返信
- 投稿と返信の編集・削除
- 画像アップロード付きの投稿 / 返信
- ノート単位のSNS / シェアシート共有
- 広い画面ではノート一覧を横に出す簡易レスポンシブ対応

## 対応予定プラットフォーム

- iOS
- Android
- macOS
- Windows
- Linux

## 開発メモ

最初の目標は、"チャットみたいに開いてすぐ書けるメモ" を最短で形にすることです。  
高機能化よりも先に、入力のしやすさ・見やすさ・続けやすさを重視します。

## セットアップ

このプロジェクトでは `mise` で Flutter を管理できます。プロジェクトルートで以下を実行します。

```bash
mise install
mise exec -- flutter pub get
mise exec -- flutter test
mise exec -- flutter run
```

初回のみ、Flutter の実行基盤を再生成したい場合は以下を使えます。

```bash
mise exec -- flutter create .
```
